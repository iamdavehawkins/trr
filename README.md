# Threads Release Radar

A minimal, retro-styled static website for the Threads Release Radar Spotify playlist.

## About

This website hosts links to Threads Release Radar, a Spotify playlist that updates every Friday with new releases from musicians of Threads.

## Features

- Clean, minimal black and white design
- Mobile-optimized layout
- Spotify playlist embedding
- Archives section for historical playlists

## Deployment

This site is deployed on GitHub Pages and accessible at [trr.iamdavehawkins.com](https://trr.iamdavehawkins.com).
